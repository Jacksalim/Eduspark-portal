# 🎓 EduSpark — Online Tutoring Portal

A full-stack tutoring portal with Supabase backend, AI-powered quizzes, video lessons, and progress tracking.

---

## 🚀 Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Set up Supabase

1. Go to [supabase.com](https://supabase.com) → **New Project**
2. Name it `eduspark`, choose **Africa (Johannesburg)** for best speed
3. Once ready, go to **SQL Editor** and paste the contents of `schema.sql` → Run it
4. Go to **Settings → API** and copy your **Project URL** and **anon public key**

### 3. Configure environment variables

Copy the example file:
```bash
cp .env.example .env.local
```

Edit `.env.local` and fill in your API keys:
```
VITE_SUPABASE_URL=https://your-project-id.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-public-key
GEMINI_API_KEY=your-gemini-api-key
ANTHROPIC_API_KEY=your-anthropic-api-key
```

> **Get API keys:**
> - **Gemini:** https://console.cloud.google.com (primary provider)
> - **Anthropic:** https://console.anthropic.com (fallback provider)
> - **Supabase:** https://supabase.com

### 4. Run the app
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 🗄️ Database Tables

| Table | Purpose |
|---|---|
| `profiles` | User accounts (learner / parent / admin) |
| `videos` | Video lessons with subject, grade, topic, URL |
| `video_watches` | Tracks which learner watched which video |
| `quiz_results` | Saves quiz scores per learner |
| `progress` | Subject progress % per learner (auto-updated) |
| `visits` | Site visitor log |

## 👤 User Roles

| Role | Access |
|---|---|
| **Learner** | Watch videos, take quizzes, view own progress |
| **Parent** | View linked child's progress and quiz scores |
| **Admin/Tutor** | Upload videos, manage learners, view all analytics |

## 🔒 Linking a Parent to a Child

In Supabase → Table Editor → `profiles`:
1. Find the learner's row and copy their `id`
2. Find the parent's row
3. Set the parent's `parent_id` column to the learner's `id`

The parent will then see that child's data in their dashboard.

---

## 🎬 Adding Videos (as Admin)

1. Sign in as an admin
2. Go to **Tutor Dashboard → Upload Video**
3. Paste a YouTube or Vimeo URL
4. Fill in subject, grade, topic
5. Click **Upload Video**

No code needed — it saves directly to Supabase and appears for learners immediately.

## 🤖 AI Quiz Generation

Quizzes are generated by a **multi-provider AI system** in real-time:

- **Primary Provider:** Google Gemini (fast, efficient)
- **Fallback Provider:** Anthropic Claude (automatic backup)

Features:
- ✅ Aligned to South African CAPS and Kenyan CBC curriculum
- ✅ Age-appropriate difficulty for grades 1–12
- ✅ Covers 5 subjects: Mathematics, English, Science, Social Studies, ICT
- ✅ Automatic provider switching if one fails
- ✅ Production-ready with error handling and retry logic

**Setup:** See [QUIZ_SETUP.md](QUIZ_SETUP.md) for detailed configuration.

## 🚢 Deployment

### Deploy to Vercel (recommended)
```bash
npm install -g vercel
vercel
```
Add your environment variables in the Vercel dashboard under **Settings → Environment Variables**.

### Deploy to Netlify
```bash
npm run build
# Drag the `dist/` folder to netlify.com/drop
```

## 📧 Email Confirmation

Supabase sends a confirmation email on sign-up by default. To disable this during testing:
**Supabase → Authentication → Settings → Email → uncheck "Enable email confirmations"**
